Graphics

What kind of photos are good to encode using JPEG?

JPEG is best for colorful photos and detailed images, not for text or sharp edges.

What kind of photos are good to encode using GIF?

GIF is best for simple images, animations, and graphics with few colors, like logos and icons.

What is unique about the PNG format?

PNG supports transparency and lossless compression, making it great for high-quality images and graphics.

How do you resize your photo to the size it should be displayed?

Open the image in Preview, go to Tools > Adjust Size, enter the dimensions, click OK, then Save. and we can aslo use CSS to adujust.

Why should you resize photos using photo editing software instead of resizing it using CSS?

Resizing photos with photo editing software reduces file size and improves load speed, while CSS only changes display size without optimizing the image, which can slow down performance.
