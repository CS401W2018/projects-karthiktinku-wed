internet

The internet is a huge network that connects computers worldwide, letting people share information, browse websites, send emails, and talk to each other.

web

The web is a collection of websites and pages that you can access on the Internet using a browser.

How are the Internet and Web different?

The internet is the global network that connects all computers and devices, while the web is a collection of websites and pages you can visit using the Internet.

ASCII and Unicode

ASCII is a system that represents basic English characters with numbers. Unicode is s a system that represents characters from all languages and symbols.

What is UTF-8

UTF-8 is a system that encodes all text characters efficiently using 1 to 4 bytes.

URL

A URL is the address you type in a browser to visit a website.

Domain Name

A Domain Name is the readable address of a website, like example.com, that helps you find it on the Internet.

HTTPS

HTTPS is a secure version of HTTP that encrypts data to keep it safe while browsing.

IP Address

An IP Address is a unique number assigned to each device on the Internet, helping to identify and locate it.

DNS

DNS It translates website names (like example.com) into IP addresses so your browser can find them.

W3C Recommendations

W3C Recommendations is to ensure that web technologies are used consistently and work well across different browsers and devices.

XML, XHTML, HTML, HTML5

XML Is A way to store and organize data. XHTML is a stricter version of HTML. HTML is The language used to create web pages. HTML5 is The latest version of HTML with extra features.
