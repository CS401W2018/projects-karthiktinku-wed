The form tag has an "action" attribute. What does it do?

The action attribute in a form> tag specifies the URL where the form data will be sent when submitted.

The form tag has a "method" attribute. What does it do?

The method attribute in a form> tag defines how the form data is sent to the server, using either GET or POST

What is the difference between a "POST" and a "GET"?

A GET request retrieves data from a server and sends parameters in the URL, making it visible and easy to bookmark. A POST request sends data securely in the request body, making it better for submitting forms or sensitive information.
