What is AJAX?

AJAX allows web pages to update dynamically without reloading by using JavaScript to send and receive data from a server asynchronously.

Why do you put your JS Code in a separate file?

Keeping JavaScript in a separate file makes the code more organized, easier to manage, and reusable across multiple pages. It keeps the HTML clean and improves readability. It also helps with faster loading since browsers can cache the script. Plus, it improves security by reducing the risk of inline script attacks.
