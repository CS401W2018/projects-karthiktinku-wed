What you learned about Flex and CSS.

CSS Grid and CSS Flexbox are both used for layout design, but they work differently. CSS Grid is a two-dimensional system, meaning it controls both rows and columns. CSS Flexbox, on the other hand, is one-dimensional, meaning it arranges elements in a single row or column

Why do you need to have an accessible website? What does that mean?

An accessible website is one that everyone, including people with disabilities, can use easily. It ensures compatibility with tools like screen readers and provides simple navigation.
