What is normal flow in HTML?

Normal flow is the default order in which HTML elements appear on the page.

What does float do? Why do you need to clear the float?

float moves an element left or right, letting content wrap around it. You clear the float to fix layout issues and ensure the container includes the floated elements.

What do absolute, fixed, and relative do?

In CSS, absolute positions an element relative to its nearest positioned ancestor. Fixed keeps an element in the same spot even when scrolling. Relative moves an element from its normal position without affecting surrounding elements.

How can you make nice navigation buttons for mobile devices?

Make navigation buttons responsive, clear, and large enough (48x48px) for easy tapping. Use media queries, icons, and high contrast for visibility. Add visual feedback on tap.
