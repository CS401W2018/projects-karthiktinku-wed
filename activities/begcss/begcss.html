CSS

Inline CSS is used within an HTML tag, embedded CSS is placed in the style tag inside the HTML document, and external CSS is in a separate file linked to the HTML.

External CSS is better because it keeps your HTML neat and easy to manage. You can use the same styles on multiple pages, making updates faster. Plus, browsers can store the CSS file, so your site loads faster.

A selector is used in CSS to target specific HTML elements so you can apply styles to them.
